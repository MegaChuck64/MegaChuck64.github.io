Cardifacts

  A 2-5D roguelike action, card-collecting game
made in Unity. It was made as my senior project
over a 25 week period. I made it with one other
programmer and two artists

  My main responsibilites included procedural
dungeon generation as well as handling the

Island Generator

  Built in Unity using Perlin Noise,
this is a procedural generating island.
I start out with a a circle with a
gradient going outwards from the center
mapping from 0-1. I then subtract each
value by the values from a perlin noise
sample, along with some other little
things to make it look prettier.

  I've added features for you to
map different colors for elements on the
island (sand, rocks, dirt, etc.) based on
ranges of those gradient perlin numbers.
I've also added the ability to save certain
islands which will be loaded when you come
back to this page.
